# Учебный автотест для API создания заказа и получения трека заказа в Яндекс.Самокате

## Описание проекта
Автоматические тесты для проверки API создания заказа и проверки, что по треку заказа можно получить данные о заказе Яндекс.Практикум в рамках обучения QA-инженера.

## Структура проекта
- `configuration.py` — URL сервиса и пути API
- `data.py` — тестовые данные пользователя и заголовки
- `sender_stand_request.py` — функции HTTP-запросов
- `test_order_by_track.py` — тест
- `README.md` — документация
- `.gitignore` — игнорируемые файлы
